# PopX Assignment

A React JS implementation of the PopX mobile application UI based on the provided design.

## Features

* Welcome Screen
* Login Screen
* Signup Screen
* Profile Screen
* React Router Navigation
* Local Storage Integration
* Mobile-Centered Responsive Layout

## Tech Stack

* React JS
* React Router DOM
* CSS3
* Vite

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
